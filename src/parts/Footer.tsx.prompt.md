# Detailed Screen Design Document

## Screen name
Footer

## Description
The Footer component will be displayed at the bottom of every page in the inventory management system. It will contain links to important pages, such as the dashboard, inventory status, and order history. It will also display the current user's role and provide a logout button.

## Screen layout
The Footer component will be a fixed position element at the bottom of the screen. It will be divided into three sections: navigation links, user information, and logout button.

The navigation links section will contain links to the dashboard, inventory status, order history, and supplier management pages.

The user information section will display the current user's role.

The logout button will allow the user to log out of the system.

## Screen behavior
The Footer component will be displayed on every page in the system. The navigation links will be consistent across all pages, allowing the user to easily navigate to important pages. The user information section will display the current user's role, which will be updated when the user logs in or out. The logout button will log the user out of the system and redirect them to the login page.

## Input Form
The Footer component will not contain any input forms.

## Error messages
The Footer component will not display any error messages.
